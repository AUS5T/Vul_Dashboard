Known Exploited Vulnerabilities (KEV)

Legend & Guide

CVSS Score: Common Vulnerability Scoring System rating, numeric severity (v4 → v3 → v2).

CVSS Severity: Descriptive level (e.g., Critical, High).

EPSS Score: Likelihood of exploitation (0.0–1.0).

EPSS Percentile: Rank among vulnerabilities (higher = more likely to be exploited).

Vulnerability (hover): NVD description with tooltip showing attack vector.
